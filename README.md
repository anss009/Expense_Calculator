# Expense Calculator

A responsive expense tracking application that allows users to manage their budget, track expenses, and view remaining balance in real time. Built using pure JavaScript with persistent storage.

## 🚀 Features
- Set and manage budget
- Add and delete expenses
- Real-time balance calculation
- Persistent data using Local Storage
- Responsive dashboard UI

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- Local Storage API
